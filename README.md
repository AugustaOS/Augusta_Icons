# SolarCut Icons

Designed for AugustaOS.

Design plan:
- 30° from the x axis is the dominant angle
- Where does the sun hit?      Ans: 30 degree from the y axis.
- Icons defined by a solar directional plane
- 30° Cut, tilt, if not 30° highlight
- Sun is at ~60° altitude at late-August in Greece
- Diamond are accepted sometimes
- Round shape = easter egg
